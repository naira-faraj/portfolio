# Naira Faraj - Interior Designer Portfolio

A beautiful, minimalist portfolio website for Naira Faraj, an interior designer. Built with Next.js 15, TypeScript, and Tailwind CSS.

## Features

- Elegant, responsive design with warm cream/beige color palette
- Hero section with polaroid-style image display
- Portfolio grid showcasing design projects
- About section with experience statistics
- Fully optimized for SEO and performance

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Customization

Replace the images in `/public/images/` with your own:

- `hero-interior.jpg` - Main hero section image
- `project-1.jpg` to `project-4.jpg` - Portfolio project images
- `about.jpg` - About section portrait

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS 4
- Google Fonts (Inter, Playfair Display)
